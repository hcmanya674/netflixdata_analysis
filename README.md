 Netflix Data Analysis 

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python.

 Tools Used
- Python
- Pandas
- Matplotlib

 Analysis Performed
- Movies vs TV Shows comparison
- Top release years
- Countries producing the most Netflix content
